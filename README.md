# Multiple Resolutions Mod
  
  <br />
  
  ‚Üî **Allows you to set 3 diffrent screen settings under the "general" settings tab.** ‚Üî  
  
  Keeps everything else vanilla
  
  ## default
   
   Keeps the default 1080p 16:9 resolution

  ## expand
   
   Scales the width and the height to viewport - so you see more in all directions üëç
   
   
  ![20221109212528_1](https://user-images.githubusercontent.com/41547570/200939663-474c9193-f84e-45d4-a257-8c7d3a6676d0.jpg)

  ## expand width
   
   Scales only the width keeping the kind of zoomed in look and fell but lets you see more on the sides ü§ù  
   <br />
   Potential Blackbars - if you don't have enough width to keep the aspect ratio.  
   2560 x 1440 -> black bars  
   4330 x 1440 -> no black bars
